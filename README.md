Присутні такі перевірки input: required та RegExp.
Якщо місто не знайдено — виводимо помилку; якщо зникає інтернет — також виводимо помилку.

При відправці форми виконуємо запит для отримання координат міста, а за координатами отримуємо погоду.

Виводимо такі дані:
- поточна температура
- мінімальна/максимальна температура
- опис погоди
- швидкість вітру
- також координати, їх можна побачити якщо натиснути на кнопку

Є історія з можливістю видалення записів. Видалення можна скасувати протягом 5 секунд (присутній прогрес-бар).

В історії, при натисканні на назву міста, відправляється форма з вибраним містом і отримується його погода.

Під час видалення елемента історії працює прогрес-бар, який показує залишок часу до видалення елементу, оновлюється лише один компонент, оскільки стан прогрес-бара є локальним.

Історія зберігається в localStorage. При повторному вході на сторінку, завдяки функції ініціалізації, значення зчитуються з localStorage і записуються в стан.

Також реалізована мемоізація всіх компонентів. Функції які є сенс кешувати, закешовані завдяки useCallback.

## Як запустити проект
1. Встановити залежності:
```bash
npm i
```
2. Створити .env файл і додати api key:
```bash
VITE_API_KEY=your_api_key
```
3. Запуск і збірка проекта:
   
Запустити в режимі розробника (запустимо на 3000 порту)
```bash
npm run dev
```
Зробити білд для продакшена
```bash
npm run build
```
